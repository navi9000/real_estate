# О проекте

Тестовое задание на позицию разработчика next.js. Необходимо вывести на экран список объектов недвижимости с возможностью переключения валют.

## Настройка и запуск проекта

Для начала необходимо создать файл .env на базе файла .env.example. Для локальной разработки значения менять не требуется. Для запуска в продакшн необходимо изменить значение переменной NEXT_PUBLIC_ROOT_URL.

npm i
npm run dev

## Особенности проекта

- Проект на next.js с использованием роутера app
- Для стилей использован tailwind и shadcn/ui
- Сохранение выбранной валюты происходит через cookie

## Примечания

Реализовал запрос относительной цены через api/currencies/{code}, т.к. в реальном проекте курсы валют менялись бы. Однако на стороне клиента я сделал простое кеширование по валюте, т.к. в условии задачи данные статичны.
